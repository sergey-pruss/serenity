Onepage: всё для топ-10 (чеклист работ)

Один документ = ТЗ на SEO и контент. Порядок: P1 → P2 → P3. Полный SERP gap (как у контекстной) — отдельным этапом после съёмки выдачи.

P1 Срочно

  1. Go-live статики + routing.conf (после ОК на конфиг): на prod сейчас legacy без H2 — главный блокер для топ-10.
  2. Title/description: в начале «разработка корпоративного сайта» и «создание корпоративного сайта»; Москва/СПб; CTA без переспама (Яндекс ~12 при 800+ показов, 0 кликов в Google).
  3. Hero: H1 «Разработка корпоративных сайтов» + подзаголовок с «создание корпоративного сайта»; полоса KPI (сайты, отрасли, сроки, награды).
  4. Два коммерческих H2: «Создание корпоративного сайта» и «Разработка корпоративного сайта под ключ» — развести интенты, не только h3 в этапах.

P2 Важно

  1. Таблица сравнения пакетов (корпоративный / интернет-магазин / лендинг) в дополнение к слайдеру price-card.
  2. Кейсы корпоративных сайтов с метриками (срок, CMS, конверсия) — блок «Ещё кейсы» уже есть, усилить SEO-текстом.
  3. FAQ 5–7 пунктов: сроки, стоимость, CMS, поддержка; FAQPage JSON-LD = видимый текст.
  4. Перелинковка: главная, /services, marketing, блог про разработку сайтов.

P3 Дополнительно

  1. После go-live — полный SERP gap (как kontekstnaya/targeting) по двум запросам × Яндекс/Google × Москва/СПб.
  2. Блок «Читайте в блоге» — 3–4 статьи про корпоративные сайты / UX / CMS.
  3. og:image и ассеты в /_sa/img/services/korporativnyj_sajt/…; переобход в GSC и Я.Вебмастер.

Порядок внедрения

P1 (сверху вниз) → P2 → P3.

Контент и техника

Контент

  • Яндекс (панель): «разработка» ~12.1 (451 пок.), «создание» ~12.1 (372 пок.) — на границе топ-10; SERP СПб: 3 и 8 (снимок 2026-05-19), Москва: 13 и 28.
  • Google: страница вне топ-20 по обоим запросам — после статики усилить title/H2 и внутренние ссылки.
  • Статическая сборка в репо: 8 H2, FAQ, кейсы, награды, price-cards — выше legacy на prod (0 H2, h1 «корпоративный сайт»).

Техника

  • Канон: https://serenity.agency/korporativnyj_sajt (без слэша).
  • Product/Offer schema на price-card — проверить цены и image на /_sa/.
  • Форма заявки в DOM (#desktop-order-popup); CWV после Nuxt CSS bundle.
  • 301 с /korporativnyj_sajt/ при go-live; sitemap lastmod после деплоя.
