Сразу после выкладки на PROD

Поверхность: явный PROD (bash scripts/deploy-prod.sh), не только коммит в git.

  1. Проверить в браузере канон и meta: serenity.agency/kontekstnaya_reklama
  2. Яндекс.Вебмастер — переобход URL страницы услуги
  3. Google Search Console — проверка URL / запрос индексирования
  4. Локально: npm run test:kontekstnaya-reklama, npm run test:service-config
  5. После prod-деплоя: npm run test:post-deploy-smoke и при необходимости выборочные проверки из team-handbook

Через 2–3 недели

  1. Снять позиции (Topvisor / rank-dashboard):
    • настройка контекстной рекламы
    • ведение контекстной рекламы
  2. GSC: клики и средняя позиция по этим запросам (раньше были показы без кликов)
  3. Не ожидать топ-20 в первую неделю после миграции на статику (май 2026) — нормальная задержка переоценки
Если позиции не двигаются после контентных блоков 1–4 на проде — смотреть SERP gap-анализ и стратегию топ-10, не править title каждую неделю.
