Команды SEO-съёмки

Запускайте в Terminal Cursor или iTerm — не через чат агента (иначе Enter не сработает).

Дашборд позиций Handbook
Капча: откроется Safari/WebKit → пройдите «Я не робот» → выберите регион в подвале → Enter в том же терминале, где бежит команда (не в чате).
Между запросами ~4–6 с; между страницами выдачи (стр. 2–5) ~8–14 с + прокрутка. Если serenity.agency уже в топе на текущей странице — следующие страницы не открываются (SERP_PAGINATE_ALL_PAGES=1 — всегда листать до топ-50). Капча часто — увеличьте: SERP_PAGE_FLIP_DELAY_MS=12000 SERP_PAGE_FLIP_JITTER_MS=8000 или SERP_CELL_DELAY_MS=6000.

Главное

Полная съёмка на сегодня (~30 ячеек)

Все страницы × запросы × Яндекс (Москва, СПб) + Google (РФ). В конце: панели GSC/Я.ВМ, сборка HTML и выкладка на static-dev.

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:serp:full

Добрать только пустые / ошибки / blocked

Не трогает уже успешные ячейки за сегодня. После съёмки — автоматически build + deploy-dev. Отключить: RANK_DASHBOARD_SKIP_DEV_DEPLOY=1.

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:serp:missing

Только панели + популярные запросы (без SERP)

Если SERP уже снят, а блок «Топ-50 за …» устарел — одной командой.

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:finish

Проверка и выкладка на dev

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run test:rank-dashboard && bash scripts/deploy-dev.sh

Панели GSC + Яндекс (без капчи)

Топ-50 популярных запросов (GSC + Я.ВМ)

Подтянуть API, собрать HTML и проверить блок «Популярные запросы» на дашборде. Нужен secrets/mcp/env.sh и GSC OAuth (seo:gsc-oauth-token:install).

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:top50

Только API панелей (без сборки HTML)

Нужен secrets/mcp/env.sh. Обычно не нужно отдельно — входит в serp:full / finish.

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:panels

Топвизор

Сравнить с эталоном и переснять расхождения

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:topvisor-diff
cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:serp:topvisor-refetch

Пересъёмка выборочно

Капча / ошибки

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:serp:refetch

Яндекс «>20»

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:serp:refetch-yandex

Сомнительные (список без браузера)

cd "/Volumes/Extreme SSD/GitHub/serenity" && npm run seo:rank-dashboard:serp:list-doubtful

Другая дата снимка

Подставьте дату вместо YYYY-MM-DD:

cd "/Volumes/Extreme SSD/GitHub/serenity" && RANK_CHECK_DATE=YYYY-MM-DD npm run seo:rank-dashboard:serp:full

Как идёт съёмка

  • Сначала все запросы по Москве (Яндекс, потом Google РФ).
  • Потом все по Санкт-Петербургу.
  • На каждой ячейке: капча → регион в подвале → Enter в терминале.

Сборка на dev: bash scripts/deploy-dev.sh → static.serenity.agency/docs/…
