Onepage: всё для топ-10 (чеклист работ)

Один документ = ТЗ на SEO и контент. Порядок: P1 → P2 → P3. Полный SERP gap (как у контекстной) — отдельным этапом после съёмки выдачи.

P1 Срочно

  1. Не каннибализировать /services/marketing: на главной — бренд «маркетинговое агентство», коммерческий блок со ссылкой «Комплексный маркетинг».
  2. Title/description: в начале «маркетинговое агентство» + Москва/СПб; description не общими фразами — УТП и 1 CTA.
  3. Первый экран: H1 или подзаголовок с «маркетинговое агентство» (сейчас H1 «Делаем маркетинг лучше» — слабая связь с ВЧ).
  4. Полоса KPI / цифры доверия над сгибом (как у лидеров выдачи по бренду).

P2 Важно

  1. Блок услуг: якоря на /services/marketing, /kontekstnaya_reklama, /kompleksnoye-prodvizheniye (после переноса).
  2. Кейсы и награды — видимы без глубокого скролла.
  3. Organization JSON-LD уже есть — проверить sameAs, logo на /_sa/.

P3 Дополнительно

  1. После правок — 4–6 недель на переоценку (статика с начала мая 2026); не менять H1 каждую неделю.
  2. Слайдер блога — анкоры под «маркетинговое агентство» / кейсы, не «читать ещё» без контекста.

Порядок внедрения

P1 (сверху вниз) → P2 → P3.
