Slack: утренние сообщения

Бот публикует короткое вдохновляющее сообщение в канал по будням в 09:00 МСК. Запуск — GitHub Actions; код в integrations/slack/.

1. Slack App (один раз)

  1. api.slack.com/apps → Create New App → From scratch (например «Serenity Morning»).
  2. OAuth & Permissions → Bot Token Scopes: chat:write; при необходимости chat:write.public для публичного канала без инвайта.
  3. Install to Workspace → скопировать Bot Token (xoxb-…).
  4. В канале: /invite @Serenity Morning (имя бота как в App).
  5. Channel ID: канал → About → внизу, или из URL (C01234ABCDE).

2. Секреты GitHub Actions

Репозиторий Settings → Secrets and variables → Actions:

Secret Значение
SLACK_MORNING_BOT_TOKEN xoxb-…
SLACK_MORNING_CHANNEL_ID C…
SLACK_MORNING_LLM_API_KEY ключ OpenAI (или совместимого API)
SLACK_MORNING_LLM_MODEL опционально, по умолчанию gpt-4o-mini

Локальный шаблон: integrations/slack/secrets.example.env → secrets.env (не коммитить).

3. Проверка и расписание

  • Workflow: Slack Morning Inspiration → Run workflow.
  • Cron: 0 6 * * 1-5 (UTC) = 09:00 МСК, пн–пт.
  • При сбое LLM уходит запасная фраза из integrations/slack/fallback-quotes.ru.json.
Смена времени (GitHub cron только UTC)
МСК cron (UTC)
08:00 0 5 * * 1-5
09:00 (сейчас) 0 6 * * 1-5
10:00 0 7 * * 1-5

4. Локальный запуск

cp integrations/slack/secrets.example.env integrations/slack/secrets.env
# отредактировать secrets.env
set -a && source integrations/slack/secrets.env && set +a
python3 integrations/slack/send_morning_message.py

Только fallback (без LLM):

SLACK_MORNING_FORCE_FALLBACK=1 python3 integrations/slack/send_morning_message.py

5. Редактирование тона

  • Системный промпт: integrations/slack/morning-prompt.md
  • Запасные фразы: integrations/slack/fallback-quotes.ru.json

После merge в main workflow подхватывает изменения автоматически. Справочник доступен на dev после bash scripts/deploy-dev.sh: /docs/slack-morning-agent.html.
