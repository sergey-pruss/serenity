Onepage: всё для топ-10 (чеклист работ)

Один документ = ТЗ на SEO и контент. Порядок: P1 → P2 → P3. Полный SERP gap (как у контекстной) — отдельным этапом после съёмки выдачи.

P1 Срочно

  1. Сниппет: title 50–60 симв., в начале «услуги комплексного маркетинга» или «агентство комплексного маркетинга»; description — Москва/СПб, кейсы, CTA без переспама (сейчас 0 кликов при позициях 1/13 в Google).
  2. Hero: подзаголовок с явным «агентство комплексного маркетинга»; полоса KPI (проекты, отрасли, NPS/лиды).
  3. H2 «Что входит в комплексный маркетинг» — перечень каналов со ссылками на /kontekstnaya_reklama, /targeting, SEO, бренд.
  4. Таблица пакетов / сравнение «Старт — Рост — Премиум» (не только слайдер карточек).

P2 Важно

  1. Отзывы и логотипы клиентов — отдельно от блока кейсов.
  2. Кейсы с метриками (рост лидов, CPL, доля каналов).
  3. FAQ: сроки, бюджет, как строится стратегия; JSON-LD FAQPage = видимый текст.
  4. Внутренние ссылки: главная → marketing; marketing → комплексное продвижение (после переноса).

P3 Дополнительно

  1. Блок «Читайте в блоге» — 3–4 статьи про комплексный маркетинг / digital.
  2. og:image на /_sa/, не admin/wp-content.
  3. Переобход URL в GSC и Я.Вебмастер после правок meta.

Порядок внедрения

P1 (сверху вниз) → P2 → P3.

Контент и техника

Контент

  • H1 сейчас «Комплексный маркетинг» — ок для бренда страницы; ВЧ лучше дублировать в первом H2 и description.
  • Страница на статике — сохранить скорость: не плодить CSS, поднять ?v= после правок stack.

Техника

  • Канон: https://serenity.agency/services/marketing (без слэша).
  • Проверить дубли title с «— Услуги — Serenity» в сниппете — упростить для CTR.
