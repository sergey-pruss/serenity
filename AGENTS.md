# Обязательные правила

- По каждой задаче писать реальные тесты; задача сдается только после успешного прохождения тестов по всем условиям задачи и контексту; после принятия задачи и фиксации результата в коммите временные тесты удалять.
- В реализации ориентироваться на оригинал `https://serenity.agency/` как на основной источник правды.
- При верстке по оригиналу (`https://serenity.agency/`) или по скриншотам пользователя добиваться 100% pixel-perfect; доводить результат самостоятельными проверочными снимками до полного визуального совпадения.
- Ничего не придумывать от себя: действовать строго в рамках оригинала; если что-то неясно, сначала уточнять.
- Сообщения коммитов писать на русском языке.
- После завершения задачи публиковать изменения сразу в двух местах: `static.serenity.agency` и Cloudflare Workers.
