Статическое зеркало serenity.agency

Вёрстка не собирается вручную: в publish/index.html — снимок с оригинала (Playwright). Для статики (/_nuxt, шрифты, favicon) подставляются полные URL на serenity.agency, без <base> — иначе Nuxt уводил бы на прод при открытии с localhost.

Выполните в корне репозитория:

npm install
npx playwright install chromium
npm run capture:home
npm run serve:publish

После запуска откройте http://localhost:4174/ (порт 4174, чтобы не пересекаться с частым python3 -m http.server 4173).

Если порт занят, задайте свой: npx --yes serve publish -l 4175 или освободите порт: lsof -nP -iTCP:4174 | grep LISTEN.

Снять с локального Nuxt (и переписать URL в прод для деплоя):

CAPTURE_URL=http://127.0.0.1:4333/ CAPTURE_REWRITE=1 npm run capture:home

Предыдущая ручная вёрстка сохранена в legacy/hand-built-index.html.
